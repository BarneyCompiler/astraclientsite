01 — Features

THREE THINGS
NO ONE ELSE
HAS DONE.

Astra Client 2 brings true social gameplay and advanced building tools to Eaglercraft. Features the community has been asking for since Eaglercraft launched.

01
World First
FRIENDS LIST

Add friends across any server. See real-time status, which server they're on, and join instantly. Never existed in Eaglercraft before now.

02
World First
PRIVATE MESSAGING

DMs and group chats that follow you across every server in real time. All messages are encrypted and secure.

03
World First
SCHEMATICS

Import custom .aschem files directly in-game. Preview, rotate, mirror, paste. WorldEdit-level power in your browser. First time ever.

02 — Schematic Hub

COMMUNITY
CREATIVE SHIELD.

Share your builds securely. Astra Client 2's locally run parser renders .aschem structural data into snapshots directly on your computer. NSFW or spoofed image uploads are physically impossible.
